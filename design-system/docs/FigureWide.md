The wide inline figure — breaks past the reading measure for the image that is the section's payoff. Everything else belongs in a MarginFig. WebP `src` with an optional original-format `fallback` (renders the site's `<picture>` markup). Inside an Essay only.

```tsx
<FigureWide
  src="/images/uat-storyboard.webp"
  fallback="/images/uat-storyboard.png"
  alt="A six-panel comic of the UAT agent walking the app."
  caption="The whole flow, one frame per step."
/>
```
