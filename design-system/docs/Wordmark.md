The logo: the site name in bold Work Sans with the brand-yellow dot (`#f9bc23`). A link when given an `href`, plain text otherwise. `SiteHeader` places it top-left on every page.

Image forms live in `assets/`: `favicon.svg` (the square "a." mark) and `ajency-wordmark.png` / `ajency-wordmark-white.png` (the full wordmark, transparent / white background) for places that need an image — email footers, social profiles, third-party tools.
