The reading column every long-form page is built in. All essay typography — kicker, pullquote, highlight, links, lists, inline `code` — is scoped to it: those pieces render unstyled outside an Essay.

```tsx
<Essay>
  <Kicker>Now</Kicker>
  <p>I got AI-pilled, spent weeks inside Claude Code, and took a project from discovery to delivery. Solo.</p>
</Essay>
```
